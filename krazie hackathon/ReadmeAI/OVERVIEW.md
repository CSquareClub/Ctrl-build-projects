## 🚀 How to Get Started

### Step 1: Run Setup (Complete in 10 minutes)
```bash
cd ReadmeAI
./setup.sh                    # macOS/Linux
# OR
setup.bat                     # Windows
```

### Step 2: Start Services
```bash
# Terminal 1 (Frontend)
cd frontend && npm run dev

# Terminal 2 (Backend)
cd backend && python app.py
```

### Step 3: Open in Browser
Visit: **http://localhost:3000**

### Step 4: Generate Your First README
1. Paste a GitHub URL (e.g., https://github.com/facebook/react)
2. Select a tone
3. Click "Generate README"
4. Copy or download your README!

## 🎓 Quick Reference

### Frontend Commands
```bash
cd frontend                    # Enter frontend folder
npm install                    # Install dependencies
npm run dev                    # Start dev server
npm run build                  # Create production build
```

### Backend Commands
```bash
cd backend                     # Enter backend folder
python -m venv venv            # Create virtual env
source venv/bin/activate       # Activate venv (Linux/Mac)
venv\Scripts\activate.bat      # Activate venv (Windows)
pip install -r requirements.txt # Install dependencies
python app.py                  # Start server
```

### API Calls
```bash
# Test the API
curl http://localhost:5000/health

# Generate README
curl -X POST http://localhost:5000/generate \
  -H "Content-Type: application/json" \
  -d '{
    "repo_url": "https://github.com/facebook/react",
    "tone": "Professional"
  }'
```

## 🐛 Troubleshooting Quick Links

| Issue | Solution |
|-------|----------|
| Port in use | Change FLASK_PORT or npm port |
| Dependencies missing | Run npm install or pip install |
| CORS errors | Check backend and frontend ports |
| Rate limit | Add GitHub token to .env |
| Module not found | Reinstall dependencies |
| Build errors | Check Node.js and Python versions |
