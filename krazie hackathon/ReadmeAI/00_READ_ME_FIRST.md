## 🚀 How to Get Started

### Option 1: Automated Setup (Recommended)

**macOS/Linux:**
```bash
cd ReadmeAI
chmod +x setup.sh
./setup.sh
```

**Windows:**
```bash
cd ReadmeAI
setup.bat
```

### Option 2: Manual Setup

**Terminal 1 (Frontend):**
```bash
cd ReadmeAI/frontend
npm install
npm run dev
```

**Terminal 2 (Backend):**
```bash
cd ReadmeAI/backend
python -m venv venv
source venv/bin/activate    # Windows: venv\Scripts\activate
pip install -r requirements.txt
python app.py
```
